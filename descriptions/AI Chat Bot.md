### AI Chat Bot

In-game public chat replies using your own AI API key (Gemini, ChatGPT, DeepSeek, or custom).

## Features
- Natural OSRS-style replies (short, casual, abbreviations)
- Conversation memory per player
- Ignore list, interaction limits, and cooldowns
- Optional Discord webhook notifications
- Conversation logging

## Setup
1. Get an API key (e.g. Google AI Studio)
2. Start your main skilling/PVM script first
3. Press **New** in Simba, open this script, and start it
4. Enter your API key / service in the GUI

## Notes
- Designed to run alongside another script in a second tab
- Keep responses in character — it will not claim to be AI
