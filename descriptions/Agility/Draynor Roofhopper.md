### Draynor Roofhopper

Runs the **Draynor Village** rooftop agility course.

## Features
- Full course obstacle handling
- Mark of grace pickup
- Optional stop-at-level
- Discord webhook on terminate
- Mouse teleport / remote input options via GUI

## Requirements
- Access to the Draynor rooftop course
- Suitable Agility level for the course

- Marked as early/testing — expect occasional edge cases
- Configure webhooks and options in the GUI
