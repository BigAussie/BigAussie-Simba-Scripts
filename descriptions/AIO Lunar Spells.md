### AIO Lunar Spells

Automates common Lunar spellbook training methods with GUI settings and optional shopping helpers.

## Spells Supported
- Bake Pie
- Humidify
- Spin Flax
- Superglass Make
- Tan Leather
- String Jewellery
- Plank Make
- Recharge Dragonstone

## Features
- Script speed presets (Slow / Normal / Fast / Turbo)
- Optional buy runes / raw materials from the GE
- Optional sell finished noted items
- Optional birdhouse runs (CanadianJames include)
- Stop-at-level support
- Discord webhook notifications
- WaspLib antiban

## Requirements
- On the **Lunar spellbook** with the spell unlocked
- Staff equipped and required runes available
- Item to process at the **top of bank tab 1** (or inventory as noted in script)
- If using a rune pouch, start with it in inventory

## Tips
- GE buy/sell uses above/below market prices — unfilled orders are not auto-cancelled
- Prefer the GUI for most settings
